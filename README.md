# Cancer Genomics Analysis - Breast Cancer Case

This project simulates a clinical cancer genomics workflow using a breast cancer case.

## Project Overview
In this project, I performed a simplified clinical-style analysis of tumor genomic data (NGS simulation).  
The goal was to identify clinically relevant mutations and interpret their impact on cancer behavior and treatment.

## What I did
- Cleaned and organized mutation data
- Selected clinically relevant genes
- Interpreted genomic alterations
- Linked mutations to potential treatment strategies
- Created a clinical-style genomics report

## Key Findings
- **TP53**: associated with aggressive tumor behavior  
- **PIK3CA**: may be relevant for targeted therapy (PI3K inhibitors)  
- **ERBB2**: amplification suggests eligibility for HER2-targeted therapy  
- **GATA3**: associated with luminal breast cancer subtype  
- **PTEN**: loss of tumor suppressor function  

## Files in this project
- `cancer_genomics_report_human.pdf` → clinical genomics report  
- `clinical_table.csv` → structured summary of key findings  

## Conclusion
This project demonstrates how genomic data can be interpreted in a clinical context to support personalized cancer treatment decisions.
